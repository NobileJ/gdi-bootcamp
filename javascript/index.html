Exercise 1

  • Create a list of items: each item has a name, a description, and a cost
  • Create a function called printItemDetails which will take an item and log to the console its name, description, and cost in the format of (for example) "$10 - Pizza - Delicious pizza, what more could you want?"
  • Loop over the list of items and call printItemDetails on each one
